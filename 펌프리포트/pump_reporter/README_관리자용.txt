========================================================
  JAEWOO Pump Intelligence System - 관리자 설치/운영 안내
  버전: v4.5.1
========================================================

1. 설치 방법
   - PumpReporter 폴더를 원하는 위치에 복사합니다.
   - 별도 설치 과정은 없습니다 (포터블 실행).

2. 실행 방법
   - PumpReporter.exe를 더블클릭합니다.
   - 최초 실행 시 필요한 폴더가 자동으로 생성됩니다.

3. 데이터 저장 위치
   운영 데이터는 아래 경로에 자동 생성됩니다:

     내 문서\PumpReporter\
       ├── input\          ← 분석할 엑셀 파일을 여기에 넣으세요
       ├── output\
       │   ├── reports\    ← 생성된 리포트 (주간/월간/분기/연간)
       │   ├── charts\     ← 차트 이미지
       │   └── cache\      ← 임시 캐시
       ├── data\
       │   ├── pump_reporter.db   ← 데이터베이스
       │   ├── settings.json      ← 사용자 설정
       │   └── pump_state.json    ← 펌프 상태 캐시
       └── logs\
           └── app.log     ← 실행 로그

   * 프로그램(exe)을 업데이트해도 위 데이터는 유지됩니다.

4. 프로그램 업데이트
   - 새 PumpReporter 폴더를 기존 위치에 덮어쓰기하면 됩니다.
   - 내 문서\PumpReporter\ 데이터는 영향받지 않습니다.

5. 데이터 백업
   - 내 문서\PumpReporter\data\ 폴더를 백업하면
     DB와 설정이 모두 보존됩니다.

6. 오류 발생 시
   - 내 문서\PumpReporter\logs\app.log 파일을 확인하세요.
   - 로그 파일을 개발팀에 전달하면 원인 파악이 가능합니다.

7. 초기화 방법
   - 내 문서\PumpReporter\ 폴더를 삭제하면
     다음 실행 시 초기 상태로 시작합니다.
   - 주의: DB와 설정이 모두 삭제됩니다.
