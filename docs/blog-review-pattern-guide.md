## 1. 제목 패턴 공식 10가지

### 공식 1: [숫자] + [키워드] + [감성 수식어]
```
예시:
- "블루투스 이어폰 추천 TOP 7, 가성비 끝판왕 모음"
- "보조배터리 추천 BEST 5, 2026년 인생템 총정리"
- "무선 마우스 추천 3가지, 실패 없는 선택법"
```
**핵심**: 숫자가 포함된 제목은 "정리된 정보"라는 인상을 주어 클릭률이 일반 제목 대비 36% 높음.

### 공식 2: [타겟] + [키워드] + [혜택/결과]
```
예시:
- "직장인 USB 충전기 추천, 책상 위 깔끔 정리"
- "자취생 수납용품 추천, 원룸 2배 넓게 쓰는 법"
- "학생 키보드 추천, 과제할 때 손목 안 아픈 제품"
```
**핵심**: 타겟을 명시하면 해당 그룹이 "내 얘기다"라고 느끼며 클릭률 상승.

### 공식 3: [연도/시즌] + [키워드] + [추천/비교]
```
예시:
- "2026년 C타입 케이블 추천, 고속충전 가능한 것만 골랐다"
- "여름 필수템 LED 조명 추천, 분위기 끝판왕"
- "2026 상반기 보조배터리 추천, 신제품 총정리"
```
**핵심**: 연도를 넣으면 "최신 정보"임을 암시. 네이버 검색에서 최신성 버프 획득.

### 공식 4: [문제 제기] + [키워드] + [해결]
```
예시:
- "맨날 끊기는 블루투스 이어폰? 이 3개면 해결됩니다"
- "충전 느려서 답답하다면, USB 충전기 이것만 사세요"
- "책상 위 지저분한 사람 필독, 정리함 추천 5선"
```
**핵심**: 질문형/문제 제기는 독자의 공감을 유도하여 체류 시간 증가.

### 공식 5: [솔직/경험] + [키워드] + [기간/횟수]
```
예시:
- "3개월 써보고 알려주는 무선 마우스 솔직후기"
- "1년 사용 후 재구매한 보조배터리 리뷰"
- "5개 비교해보고 고른 키보드 가성비 1등"
```
**핵심**: "직접 써봤다"는 신뢰감이 네이버 D.I.A 로직에서 높은 점수를 받음.

### 공식 6: [부정 회피] + [키워드]
```
예시:
- "실패 없는 핸드폰 거치대 추천 (이건 진짜 다릅니다)"
- "후회 없는 LED 조명 고르는 법, 이것만 확인하세요"
- "돈 낭비 막아주는 C타입 케이블 구매 가이드"
```
**핵심**: 손실 회피 심리 자극. "실패하지 않겠다"는 안도감 제공.

### 공식 7: [비교] + [키워드] + [결론]
```
예시:
- "앤커 vs 벨킨 USB 충전기, 뭘 사야 할까?"
- "에어팟 vs 갤럭시 버즈, 5가지 기준으로 비교"
- "다이소 vs 무인양품 수납용품, 가격 대비 승자는?"
```
**핵심**: 비교형은 구매 직전 단계 검색자를 타겟. 전환율이 가장 높은 제목 유형.

### 공식 8: [가격대] + [키워드] + [추천]
```
예시:
- "2만원대 블루투스 이어폰 추천, 이 가격에 이 음질?"
- "1만원 이하 C타입 케이블 끝판왕 3선"
- "5만원대 기계식 키보드 가성비 추천"
```
**핵심**: 구체적 가격대 명시는 예산이 정해진 검색자의 즉시 클릭 유도.

### 공식 9: [리스트] + [키워드] + [용도별]
```
예시:
- "용도별 보조배터리 추천 (여행/출퇴근/캠핑)"
- "사무용 vs 게이밍, 무선 마우스 용도별 추천"
- "공간별 수납용품 추천 (주방/욕실/현관)"
```
**핵심**: 용도별 분류는 검색 의도에 정확히 매칭되어 체류 시간 극대화.

### 공식 10: [강조 어미] + [키워드]
```
예시:
- "이것만 사세요, 2026 USB 충전기 추천"
- "진짜 이거 하나면 끝, 핸드폰 거치대 추천"
- "제발 이거 쓰세요, 가성비 키보드 추천"
```
**핵심**: 명령형/강조형 어미는 "전문가가 콕 집어준다"는 느낌으로 신뢰도 상승.

### 제목 작성 시 필수 규칙

| 규칙 | 설명 |
|------|------|
| 25자 이내 | 32자 넘으면 검색 결과에서 "..."으로 잘림 |
| 키워드 앞배치 | 핵심 키워드를 제목 앞쪽에 배치 |
| 키워드 2-3개 | 한 제목에 키워드는 2~3개가 적정 |
| 동일 키워드 비반복 | 같은 키워드 2번 이상 반복 금지 |
| 자연스러운 문장 | 슬래시(/) 나열형보다 자연스러운 문장이 노출에 유리 |

## 2. 글 구조 템플릿 3종

### 템플릿 A: 짧은 글 (800~1,200자) — 단일 상품 후기

```
[구조]

■ 도입 (2~3줄)
  → 개인적 상황/고민 공유
  예: "재택근무 시작하면서 충전기가 부족해서 매일 콘센트 쟁탈전이었어요."

■ 상품 소개 (핵심 정보)
  → 상품명, 가격, 구매처
  → 핵심 스펙 3~4가지 (불릿 포인트)
  [사진 1: 패키지/개봉]

■ 사용 후기 (체험 중심)
  → "직접 써보니" 로 시작
  → 장점 2~3개 (구체적 상황 묘사)
  → 아쉬운 점 1개 (신뢰도 확보)
  [사진 2~3: 실사용 모습]

■ 마무리 (2~3줄)
  → 누구에게 추천하는지 명시
  → 구매 링크 또는 검색 키워드 안내
  [사진 4: 전체 구성품]

■ 해시태그 (5~8개)
```

**적합한 경우**: 체험단 후기, 단일 제품 리뷰, SNS형 가벼운 후기

## 4. SEO 키워드 배치 가이드

### 4-1. 키워드 선정 기준

| 구분 | 설명 | 예시 |
|------|------|------|
| 대표 키워드 | 검색량 높지만 경쟁 치열 | "블루투스 이어폰 추천" |
| 세부 키워드 | 대표의 10~30% 검색량, 경쟁 낮음 | "2만원대 블루투스 이어폰 추천" |
| 롱테일 키워드 | 검색량 30 이상이면 유의미 | "자취생 가성비 블루투스 이어폰" |

**전략**: 세부 키워드로 상위노출 확보 후 → 대표 키워드 도전

### 4-2. 키워드 배치 위치별 가중치

```
[최고 가중치]
├── 제목: 핵심 키워드 1회, 앞쪽 배치 (변형 없이 정확히)
├── 첫 문장 (서론 첫 줄): 핵심 키워드 1회
│
[높은 가중치]
├── 소제목 (볼드/구분선): 세부 키워드
├── 본문 중간: 핵심 키워드 2~3회 자연 배치
│
[보통 가중치]
├── 이미지 대체 텍스트: 키워드 포함
├── 본문 후반: 유사/연관 키워드
│
[낮은 가중치]
└── 해시태그: 핵심 + 연관 + 롱테일 조합

□ 동영상: 가능하면 네이버 자체 동영상 1개 추가
□ 글 길이: 최소 800자, 권장 1500~2500자
□ 가독성: 3~4줄마다 줄바꿈, 소제목 활용, 표/불릿 포인트
```

### 주간 발행 스케줄 예시

| 요일 | 콘텐츠 유형 | 글 길이 |
|------|-------------|---------|
| 월 | 상품 추천 리스트 (표준글) | 1,500~2,500자 |
| 수 | 단일 상품 상세 리뷰 (상세글) | 2,500~4,000자 |
| 금 | 사용 팁/가이드 (짧은글) | 800~1,200자 |

> 하루 10개 몰아쓰기보다, 주 2~3개 규칙적 발행이 블로그 지수에 훨씬 유리.
