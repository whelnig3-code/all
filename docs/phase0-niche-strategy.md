# Phase 0: 니치 전략 & 상세페이지 차별화 설계

> 2026-03-10 | 도매꾹 1,468개 샘플 + 61개 키워드 경쟁 분석 기반

## 전략적 판단 근거

### 시장 현실
- 스마트스토어 위탁판매는 **레드오션 중 레드오션** (나무위키, 업계 공통 평가)
- 같은 도매꾹 상품을 파는 셀러가 수백 명 → 10원 가격 경쟁은 소모전
- 알리/테무 직구가 한국 시장 4.3조원 → 가격 경쟁만으로는 생존 불가
- **핵심**: 레드오션 안에서 "마이크로 니치"를 찾아 전문 스토어로 포지셔닝

### 니치 선정 기준
1. **반복 구매 가능성** — 일회성 구매 < 소모품/교체품
2. **스펙 검색 비율** — "전동드릴" (일반) < "드릴비트 8mm HSS" (구체적)
3. **전문 지식 진입 장벽** — 아무나 설명 못 하는 카테고리
4. **알리/테무와의 차별점** — 빠른 배송 + KC인증 + 규격 정확성이 중요한 품목
5. **가격대** — 3,000~30,000원 (충동구매 가능 + 마진 확보)

## 추천 니치: 공구 소모품 & 부속 전문

### 왜 이 니치인가

**"전동드릴을 파는 게 아니라, 전동드릴을 이미 가진 사람에게 비트를 판다"**

| 비교 항목 | 전동드릴 (본체) | 드릴비트/디스크 (소모품) |
|---------|-------------|----------------|
| 구매 빈도 | 3-5년에 1번 | 3-6개월에 1번 |
| 가격 경쟁 | 극심 (브랜드 vs 중국산) | 규격만 맞으면 구매 |
| 상세 스펙 중요도 | 중간 | **극히 높음** (사이즈, 재질, 호환성) |
| 알리/테무 위협 | 높음 (14일 기다려도 OK) | **낮음** (당장 필요, 규격 확인 어려움) |
| 재구매율 | 거의 없음 | **높음** (닳으면 교체) |
| 상세페이지 차별화 | 어려움 | **쉬움** (호환표, 사용 가이드) |

### 세부 니치 구성

#### 1차 니치: 전동공구 소모품 (메인)
- 드릴비트 (HSS, 코발트, 초경, 스텝드릴, 홀쏘)
- 연마디스크 (절단, 연마, 플랩, 와이어)
- 절단날 (원형톱날, 직소날, 금속절단)
- 샌딩페이퍼 / 벨트
- 충전배터리 / 충전기 (호환품)

**도매꾹 공급**: 120+ 상품 (드릴비트 30, 연마디스크 30, 절단날 30, 기타)
**도매가 범위**: 300원 ~ 25,000원 (중위가 5,000~10,000원)
**예상 판매가**: 3,000원 ~ 40,000원 (마진 30-50%)

#### 2차 니치: 측정 & 안전 장비 (보조)
- 줄자, 수평기, 캘리퍼스, 각도기
- 작업장갑, 보안경, 안전모, 방진마스크
- 공구함, 공구가방, 부품함

**도매꾹 공급**: 210+ 상품
**도매가 범위**: 600원 ~ 24,000원

### 스토어 컨셉

**스토어명 방향**: "[상호] 공구자재" 또는 "[상호] 프로공구"
**핵심 메시지**: "전동공구 소모품 전문 — 규격 정확, 당일 출고"

### 회피 카테고리

| 카테고리 | 회피 이유 |
|---------|---------|
| 캠핑용품 | 10조 시장이지만 경쟁 극심, 계절 변동 큼 |
| 전자 케이블/젠더 | 알리/테무 직구 최적 품목, 가격 경쟁 불가 |
| 생활/주방 | 차별화 포인트 없음, 일반 소비자 대상 |
| 반려동물 | 식품/접촉 이슈, 전문성 없음 |
| 의류/패션 | 도매꾹 품질 이슈, 반품률 높음 |

## 상세페이지 차별화 전략

### 현재 vs 목표

**현재 상세페이지**:
```
[이미지] → [스펙 테이블] → [배송 정보] → [반품 정보]
```
이건 모든 셀러가 하는 기본. 고객 입장에서 **"왜 여기서 사야 하지?"**에 대한 답이 없다.

**목표 상세페이지**:
```
[차별점 배너] → [구매 가이드] → [이미지] → [호환성 표] →
[비교표] → [스펙 테이블] → [FAQ] → [배송/반품] → [관련 상품]
```

### 각 섹션 상세

#### 1. 차별점 배너 (최상단, 가장 중요)
```html
<div style="배경색 강조">
  ✅ 규격 100% 정확 보장 — 틀리면 교환+반품비 무료
  ✅ 평일 오후 2시 전 주문 → 당일 출고
  ✅ KC 안전 인증 제품
  ✅ 상세 규격표 제공 — 구매 전 호환 확인 가능
</div>
```
이 배너 하나가 전환율을 2배 올릴 수 있다.

#### 2. 구매 가이드 (전문성 시그널)
카테고리별 자동 생성:
- **드릴비트**: "어떤 드릴비트를 골라야 할까? — 재질별 용도 가이드"
  - HSS: 일반 금속, 나무 / 코발트: 스테인리스, 경합금 / 초경: 콘크리트, 석재
- **연마디스크**: "절단 vs 연마 vs 플랩 — 어떤 디스크가 필요한가요?"
- **절단날**: "원형톱날 이빨 수에 따른 차이 (24T vs 40T vs 60T)"

이런 콘텐츠는 **LLM으로 카테고리별 템플릿을 생성**할 수 있다.

#### 3. 호환성 표 (핵심 차별점)
```
이 드릴비트는 다음 전동드릴과 호환됩니다:
✅ 보쉬 GSB/GBH 시리즈
✅ 마키타 HR/HP 시리즈
✅ 디월트 DCD/DCH 시리즈
✅ 계양/아임삭 일반 13mm 척
```
알리/테무에서는 절대 제공하지 않는 정보. **한국 사용자가 실제로 궁금해하는 것**.

#### 4. 비교표 (같은 카테고리 내 상품 비교)
```
| 모델 | 지름 | 재질 | 용도 | 가격 |
|------|------|------|------|------|
| A형  | 6mm  | HSS  | 나무/일반금속 | 3,900원 |
| B형  | 6mm  | 코발트 | 스테인리스 | 6,900원 |
| C형  | 6mm  | 초경  | 콘크리트 | 9,900원 |
```

#### 5. FAQ (검색 유입 + 신뢰)
LLM으로 카테고리별 자동 생성:
- "이 비트가 제 드릴에 맞나요?"
- "HSS와 코발트 비트 차이가 뭔가요?"
- "한 세트에 몇 개 들어있나요?"
- "내구성은 어느 정도인가요?"

### 구현 방식

기존 `detail-content-builder.ts`를 확장하여:
1. **카테고리 감지** → 공구 소모품이면 전문 템플릿 적용
2. **구매 가이드 생성** → 카테고리별 사전 정의 콘텐츠 맵
3. **호환성 표** → 도매꾹 API 옵션 데이터 + 규격 패턴 매칭
4. **FAQ** → 카테고리 + 스펙 기반 LLM 생성 또는 사전 정의
5. **차별점 배너** → 모든 상품 공통 (스토어 브랜딩)
